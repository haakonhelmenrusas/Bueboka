[![Get it on Google Play](./github_assets/PlayStore.png)](https://play.google.com/store/apps/details?id=com.aaronshade.bueboka&hl=no_nb)
[![Download on the App Store](./github_assets/AppStore.png)](https://apps.apple.com/no/app/bueboka/id6448108838?l=nb)

**Bueboka** is a comprehensive archery tracking application for iOS, Android, and Web. Track your practice sessions, manage equipment, and improve your skills with detailed statistics and offline support.

## Features

- Practice tracking with scores, distances, and weather conditions
- Equipment management for bows and arrow sets
- Offline support with automatic sync
- Secure authentication (email, Google, Apple)
- Data visualization and statistics
- Ballistics calculator for sight marks

## Tech Stack

Built with **React Native**, **Expo SDK 55**, **TypeScript**, and **Expo Router**.

## Prerequisites

- Node.js (v18+)
- Expo CLI
- Android Studio or Xcode for native development

## Getting Started

```bash
# Clone and install
git clone https://github.com/Aaronshades/Bueboka-app.git
cd Bueboka-app
npm install

# Configure environment
echo "API_URL=your_api_url_here" > .env

# Start development
npm start

# Run on platform
npm run ios      # iOS
npm run android  # Android
npm run web      # Web
```

## Available Scripts

- `npm start` - Start development server
- `npm test` - Run tests
- `npm run lint` - Lint code
